# Python-Week-1
Overview
The Campus Expansion Planner is a user-friendly desktop application built with Python's Tkinter library that helps university administrators and facility planners manage building expansion projects efficiently. This application provides an intuitive interface for tracking building specifications, costs, and generating reports for campus development planning.

**Key Features**
Building Management
Add New Buildings with comprehensive details including:

Building name

Area (square feet)

Number of floors

Building type (Academic, Admin, Residential, Lab, Library)

Estimated cost ($)
Data Visualization
Interactive List View displaying all buildings in a sortable table format

Real-time Statistics showing total number of buildings and cumulative costs

Detailed Reports with complete building information and summary statistics

🔧 Administrative Tools
Delete Individual Buildings with confirmation dialog

Clear All Buildings for starting fresh

Sample Data pre-loaded for demonstration purposes

Export to CSV for external analysis and reporting
Technical Stack
Language: Python 3.x

GUI Framework: Tkinter (built-in Python library)

Data Storage: In-memory list structure

File Export: CSV format
Installation & Usage
Prerequisites
Python 3.6 or higher

Tkinter (comes pre-installed with Python)
How It Works
Adding Buildings: Fill in the building details in the left panel and click "Add Building"

Managing Buildings: Select any building from the list to delete it

Generating Reports: Click "Show Report" for a detailed summary

Exporting Data: Use "Export Data" to save all information as a CSV file
